# BASE CROWN

## Overview
BASE CROWN is a fully onchain strategy and progression game built as a Base Mini App.
Players collect Crown Fragments, forge Relics, complete quests, attack rivals, climb leaderboards and compete for rewards.
All game state is stored on Base. No centralized database is required.
The game economy revolves around CROWN, Fragments, Keys and Relics.
Players can play for free, while Genesis Crown Key holders receive additional rewards and daily advantages.
The objective is to grow stronger, forge better Relics, earn higher rankings and compete for seasonal rewards.

## Core Loop
1.Claim daily Spins and Attacks.
2.Use Spins to earn Fragments, Keys, CROWN and other rewards.
3.Use Attacks to steal resources from other players.
4.Complete Quests for additional rewards.
5.Collect Fragments and CROWN.
6.Forge Relics using Fragments and CROWN.
7.Relics provide bonuses and progression advantages.
8.Climb the leaderboard through activity and score.
9.Earn seasonal rewards.
10.Repeat daily.

## Spins
Daily Spins

Normal Players:

- 5 Free Spins per day

Genesis Crown Key Holders:

- 10 Free Spins per day

Daily Spins reset every 24 hours.

Unused Spins do not carry over.

Extra Spins

Players can purchase additional Spins using ETH.

Extra Spins cannot be purchased using CROWN.

Reason:
Prevent players from wasting large amounts of CROWN on luck-based mechanics.

Spin Rewards

Possible Rewards:

- CROWN
- Crown Fragments
- Crown Keys
- Extra Attack
- Extra Spin
- Crown Chest

Reward Probabilities

Common Rewards:

- CROWN
- Fragments

Uncommon Rewards:

- Keys
- Extra Attacks

Rare Rewards:

- Crown Chests

Future updates may introduce:

- Rare Relics
- Event Tickets
- Limited Collectibles

Spin Cooldown

No cooldown between Spins.

Players may use all available Spins immediately.

Spin Security

All reward generation is performed onchain.

Spin results are recorded on Base.

Purpose

Spins provide the primary daily progression loop.

Players use Spins to collect resources required for:

- Relic Forging
- Relic Upgrades
- Event Participation
- Leaderboard Progression
